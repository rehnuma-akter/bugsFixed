How does JavaScript work?

JavaScript is an interpreted programming language that runs at the client-side or on the client machine inside a browser. Presently all web browsers support JavaScript codes with the help of their JavaScript engines. So whenever a JS code is written and run, the browser loads the web page, parsing HTML it creates the DOM which is the live view of the page contents that can be accessed through JavaScript. Then the CSS is loaded. After the HTML and CSS is put together the JS engine loads the JS files but does not immediately run codes . It lets HTML and CSS to load first, when done JS code is executed from top to bottom order.

What are the differences between local storage and session storage?

Local storage does not expire but session storage is cleared when the page session ends. The data in session storage is persisted only until the window or tab is closed. But in local storage the data is persisted until the user manually clears browser cache or the web page clears the data. Both local and session storage can store only strings. On some browser like safari(8.03) local storage has a limit of 2551 k characters but session storage is unlimited in storage.
